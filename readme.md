# 对应任务任务自动生成学情报告与分析
使用技术node.js=10.8.2 
## 初始化并下载echars依赖 

### 初始化

~~~
npm init -y
~~~

### 安装echars依赖

~~~
npm install echarts 
~~~

### 启动项目

~~~
npx serve .
~~~

## 



## 文件说明

data.json存储数据

index.html是展示页面
